## Cat-3

# What It Is

A simple website where you look at cute cats and also learn fun facts (Rhymes). 

# Why I Made It

I saw a Tiktok video about APIs and got motivated to make something integrating it and made it about cats because I saw a cute ginger cat after that video.

# How I Made It

Research all cat related APIs I could find.

Tech Stack:

- HTML
- JavaScript

APIs used:

- Cataas API 
- Catfact.ninja

# What I Struggled With & What I Learnt

As this was a simple project, I didn't really struggle. The Next button had a bit of a bug so I added the feature that makes it stop while an image is loading.

A simple project can be really fulfilling too :)
